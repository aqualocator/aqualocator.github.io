Set Webex as Your Default Calling App

If you’d like links like tel:4595 to open automatically in Webex instead of another dialer:

  1. Open the Windows Settings app.
  2. Go to Apps → Default Apps.
  3. Search for Webex in the list.
  4. Scroll down and find the URL:tel or Telephone protocol.
  5. Click it and choose Webex as the default app.

Once done, any extension link (e.g., tel:4595) will open directly in Webex if it’s running.

Tip: The first time you click a phone link, Windows may prompt you to select a default app — choose Webex and check “Always use this app.”
