Set Up Webex Calling

Follow these steps to register Webex as your default calling app for employee extensions.

Requirements for Setup

  1. Confirm Webex app is open and you're logged in successfully.
  2. Enter a valid 4-digit extension below and click Call Test.
  3. When prompted, click “Open Pick an app”.
  4. Select Webex from the suggested apps list.
  5. Click “Always” to set Webex as your default calling app.
  6. Check “Always allow this type of link” and click “Open Webex”.

Once Webex is set as default, extensions like Ext: 1234 will open directly in Webex.

If the pop-up doesn’t appear or Webex doesn’t open,
please contact the IT Helpdesk at ITHelpdesk@aqua-aerobic.com
